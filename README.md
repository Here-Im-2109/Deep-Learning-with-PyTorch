# Deep-Learning-with-PyTorch


## Workflows

1. Create a vertual env in conda.
2. Activate the env.
3. Run the CODES.

# How to run?
### STEPS:

Clone the repository

```bash
git clone https://github.com/Here-Im-2109/Deep-Learning-with-PyTorch.git
```
### STEP 01- Create a conda environment after opening the repository

```bash
conda create -p pytorch_venv python=3.10.0 -y
```

```bash
conda activate pytorch_venv
```

```bash
pip install ipykernel
```
### Select the code editor (VS Code) and kernel to run all the ipynb files
